# 飞书多维表格 - 任务委托单导出插件

将多维表格选中行的数据导出为 Word 文档。

## 功能

- 读取选中行的字段：任务名称、委托单号、期望完成日期、制料阶段、制样阶段、检测阶段、附件
- 生成格式化的 Word 文档（.docx）
- 自动下载生成的文件

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 使用方法

1. 在飞书多维表格中添加此插件
2. 选中包含任务数据的一行记录
3. 点击"导出选中行到 Word"按钮
4. 自动下载生成的任务委托单文档

## 字段要求

多维表格需要包含以下字段：
- 任务名称
- 委托单号
- 期望完成日期
- 制料阶段
- 制样阶段
- 检测阶段
- 附件
